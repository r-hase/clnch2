    &HiddenCommand で予約したコマンドを無効にして、通常モードに戻す内部命令
    &ClearHiddenCommand を追加

    無効にするようにした ( &AllowTypeMiss 相当 )

    擬似キーストロークを生成する内部命令 &Key、&KeyDown、&KeyUp を追加

    コマンドの実行を指定した時間だけ止める内部命令 &Sleep 追加

    エラーをログに残すようにした ( &PopLog で確認可能 )

    コマンド入力部にはエラーを出さないようにし、入力内容を保つようにした

    複数の引数は分解して履歴に残すオプションを追加

    リスト自動表示を一時的に無効にする内部命令 &AutoListOff を追加

    &TypeMissCancelOff を解除する &TypeMissCancelOn を追加
    &AutoCompleteOff を解除する &AutoCompleteOn を追加
    &AutoFixOff を解除する &AutoFixOn を追加
    &AutoListOff を解除する &AutoListOn を追加

    日本語の曜日表現に置換する置換文字列 %k を追加


  [調整]

    パスの区切りに [ / ] を使う、設定の時、履歴には \ を / に置換して残すよ
    うにした

    ファイラによるコマンドは残さない、オプションがいつのまにか常に On の動作
    になっていたのが判ったが、誰も困っていなかったので廃止

    単語区切り文字の入力による、適当な位置まで確定、の機能において、\ と /
    を同一視するようにした

    %FullPath[*] は、[ ] 内のファイルパスをフルパスに変換したものに置換され
    る &Pwd を廃止 (必要であれば &TextOut;%pwd が使用できる)

    &FixAll を廃止 (必要であれば &SetCaret;-1 が使用できる)

    情報表示機能の項目数を３つから１つに絞った
        ( ２つ目以降は &TextOut で代用していただく方針 )

    パスの区切りに [ / ] を使う、設定の時、ファイラから渡されるファイルパス
    の [ \ ] を自動的に [ / ] に変換するようにした

    スペースを含み、まだ存在しないファイル名を、新規作成等の目的でアプリに
    渡す際に、自動 Quote 処理が効かなかったので、ディレクトリが存在すれば、
    スペースを含むファイル名であると認識するようにした

    指定したの登録コマンドの Link 先に置換される特殊変数 %Link[] 追加

    コマンド設定ツールに、ソート機能をつけた

    clnchset.exe を clnchcmd.exe と改名した
        ( clnchset.exe は 削除してください )

    clnch.dat を clnch.opt と改名した
        ( clnch.opt が生成されたら clnchset.exe は 削除してください )

    &Setup は &SetCommand に改名

    &AllowTypeMiss を &TypeMissCancelOff に改名

    プロポーショナルフォントで、リスト表示の横幅が狭くなることがあったので
    調整

    %% を % に置換するのは、最後の一度だけにした

    %LinkDir など [] で引数をとるものは、まず [] の中を置換するようにして、
    ネスト出来るようにした

    のテキストがさらに時計などの情報に置換される不具合を修正

    ホットキーを押してもアクティブ化に時間がかかる時があった不具合を修正

    全角文字の削除が変だったのを修正

    &WinActivate を使ったときにリストが表示されていると、他のウインドウがア
